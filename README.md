Simple Denial Of Service Attack in python 


Run this in powershell to install it, Or you can Install It manually !

Run this if you have/use onedrive: powershell -NoProfile -ExecutionPolicy Bypass -Command "iwr https://raw.githubusercontent.com/Soggaling/DOSAttack/refs/heads/main/DOSAttack.py -OutFile $env:USERPROFILE\Onedrive\Desktop\DOSAttack.py"

Run this if you dont have/use onedrive: powershell -NoProfile -ExecutionPolicy Bypass -Command "iwr https://raw.githubusercontent.com/Soggaling/DOSAttack/refs/heads/main/DOSAttack.py -OutFile $env:USERPROFILE\Desktop\DOSAttack.py"
